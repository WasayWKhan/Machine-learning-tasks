# Machine Learning Tasks Repository

## 📌 Overview
This repository contains 4 machine learning tasks implemented using Python and scikit-learn.

---

## 📂 Tasks Included

### 🔹 Task 1: Random Forest Classifier
- Dataset: Breast Cancer (sklearn)
- Goal: Predict malignant or benign tumor

### 🔹 Task 2: Logistic Regression
- Dataset: Diabetes Dataset
- Goal: Predict diabetes (0 or 1)

### 🔹 Task 3: XGBoost Classifier
- Dataset: Titanic Dataset
- Goal: Predict survival

### 🔹 Task 4: Decision Tree Classifier
- Dataset: Pima Indians Diabetes
- Goal: Predict diabetes

---

## ⚙️ Steps Performed in All Tasks

- Data Loading
- Data Cleaning
- Handling Missing Values
- Feature Encoding
- Train-Test Split (80/20)
- Feature Scaling
- Model Training
- Model Evaluation:
  - Accuracy
  - Confusion Matrix
  - Precision, Recall, F1-score
  - ROC-AUC Score

---

## 📊 Output
All outputs are stored in `output.txt`
